J Harfi Terimler

Java

Platform bağımsız bir programlama dili.

JavaScript

Web sayfalarına etkileşim kazandırmak için kullanılan popüler bir programlama dili.

JSON (JavaScript Object Notation)

Verileri organize etmek ve iletmek için kullanılan hafif bir veri formatı.

JVM (Java Virtual Machine)

Java kodunun çalışmasını sağlayan sanal makine.

jQuery

JavaScript ile daha kolay ve hızlı web geliştirme yapmayı sağlayan kütüphane.

JIRA

Yazılım geliştirme projelerini yönetmek için kullanılan popüler bir proje yönetim aracı.

Job (İş)

Bir sistem veya uygulama içinde belirli bir görevin yerine getirilmesi.

JUnit

Java projelerinde test yazmak için kullanılan bir framework.
